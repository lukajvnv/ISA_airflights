# ISA_airflights

Repozitorijum iz predmeta Internet Softverska Arhitektura

## Uputstvo za pokretanje aplikacije

### Poktretanje front-end aplikacije:
_U glavnom folderu projekta nalazi se direktorijum frontend u kome je sadržan Angular front-end projekat. Nakon instaliranja svi neophodnih biblioteka, odnosno modula, komandom **ng serve** se pokreæe projekat. Za uspešno prikazivanje neophodno je u URL-u uneti localhost:4200._

### Pokretanje back-end aplikacije:
_Back-end aplikacija je maven projekat. Aplikacija je implementirana u Javi korišæenjem Spring frameworka. Pokreæe se iz Eclipse razvojnog okruženja. Backk-end aplikacija osluškuje na portu 8836._
